# Heart-disease-flask-api

This project is a Heart Disease Prediction system built using multiple machine learning classification algorithms to find the best performing model. Seven different algorithms were used, including **Logistic Regression, Random Forest, Decision Tree, XGBoost, Artificial Neural Network, Support Vector Machine, and Naive Bayes.** After comparing their training and testing accuracies, Logistic Regression showed the most balanced and reliable performance. To ensure the model’s stability, cross-validation was also applied. Finally, the selected Logistic Regression model was used to build a Flask API, along with a simple HTML interface, to make real-time heart disease prediction possible through a web application.
